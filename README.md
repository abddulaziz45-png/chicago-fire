# Fire & Phoenix - Great Chicago Fire 1871

Interactive digital heritage project about the Great Chicago Fire of 1871.

This work was carried out within the framework of the Digital Heritage course at Politecnico di Torino in the academic year 2025-2026.

## Features

- Interactive timeline animation
- Historical street information
- Mini-game: Save Chicago from the Great Fire
- Photo library
- Quiz section
- Multi-language support (EN, RU, IT, TR)

## Technologies

- HTML5
- CSS3
- JavaScript
- Canvas API (for game)
